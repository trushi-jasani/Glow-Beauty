## 🌟 Overview
Glow Beauty is a **modern cosmetic store website** that allows users to:
- 👩‍💻 Login & Sign Up with form validation  
- 🛍️ Explore beauty products with images and prices  
- 💅 Discover services like bridal makeup, skincare, and more  
- 📞 Contact via form, email, or social media links  
- 📱 Enjoy a fully responsive design on all devices
